# Shay Jones Project 1

This project is a blog website built using Gatsby and Contentful as part of ITDEV-164.

## Technologies Used

- React
- Gatsby
- Styled Components
- Contentful CMS
- GraphQL
- GitHub for version control

## Features

- Pulls blog post content from Contentful
- Displays posts dynamically on the homepage
- Styled using Styled Components
- Responsive blog card layout

## How to Run the Project

1. Clone the repository
2. Install dependencies

npm install

3. Start the development server

npm run develop

4. Open in browser

http://localhost:8000
